—--

Al configurar el proyecto, crea siempre un archivo vercel.json en la raíz para asegurar el despliegue

correcto de una SPA (Single Page Application) con Vite.

El archivo debe tener exactamente esta configuración:

1. framework: "vite"

2. outputDirectory: "dist/public"

3. rewrites: Una regla que redirija todas las rutas (source: "/(.*)") a /index.html para que el

routing del lado del cliente funcione sin errores 404.

No esperes a que falle el deploy, inclúyelo desde el principio.
